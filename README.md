# Gemilura: O Seu Desenvolvedor Front-end Automático!

Bem-vindo ao **Gemilura**, uma aplicação que combina a criatividade e a tecnologia para gerar páginas web incríveis usando HTML, CSS e JavaScript. Este projeto utiliza a Google Gemini API para responder a prompts fornecidos pelo usuário e gerar conteúdo dinâmico e estilizado para seus projetos web.

## Descrição

Gemilura é uma ferramenta interativa que ajuda desenvolvedores front-end a criar páginas web, designs, banners, protótipos e muito mais. Usando um sistema de prompts, os usuários podem gerar automaticamente código otimizado para SEO e totalmente responsivo. O foco do projeto é maximizar a produtividade dos desenvolvedores, fornecendo código estruturado e estilizado para diferentes cenários, desde simples banners até sites completos.

## Funcionalidades

- **Geração de Páginas HTML**: Cria código HTML a partir de prompts fornecidos pelo usuário.
- **Estilização Dinâmica**: Aplica CSS e animações diretamente no código gerado, tornando o conteúdo visualmente atrativo e responsivo.
- **Suporte a SVG e Animações**: Integra ícones SVG e animações CSS para designs modernos e interativos.
- **Interface de Edição**: Exibe pré-visualização de código e resultados no mesmo ambiente, permitindo alternar entre visualização e código.

## Como Funciona

1. O usuário insere um prompt descritivo.
2. Gemilura processa a solicitação usando a **Google Gemini API** para gerar o código necessário.
3. O código gerado é exibido e pode ser copiado ou visualizado diretamente na interface.
4. Funcionalidades adicionais, como inserção de API Keys, são suportadas para usuários que desejam conectar serviços externos.

## Instalação

Para executar o Gemilura localmente:

1. Clone o repositório:
   ```bash
   git clone https://github.com/CrimsonSunrise/Gemilura.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Execute o servidor local:
   ```bash
   npm start
   ```

## Dependências

- **Google Generative AI SDK**: Usado para gerar conteúdo baseado em prompts.
- **Marked.js**: Para conversão de markdown em HTML dinâmico.
- **Highlight.js**: Para realce de código no modo de visualização.
- **Prism.js**: Para estilização adicional do código.

## Estrutura do Projeto

- `/assets`: Contém as imagens e ícones usados na interface.
- `/utils`: Scripts utilitários para funcionalidades como API Key e animações.
- `index.html`: O arquivo principal da interface do projeto.
- `style.css`: Estilos gerais da aplicação.
- `main.js`: Lógica de interação e controle da interface.

## Como Usar

1. Inicie a aplicação e insira sua API Key do Google AI Studio para começar.
2. Forneça um prompt para gerar uma página HTML ou elemento.
3. Visualize e copie o código gerado diretamente na interface.

### Exemplos de Prompts

- "Crie uma página de login com um formulário simples e botões estilizados."
- "Gere um site de portfólio com três seções: Sobre, Projetos e Contato."
- "Crie uma página de e-commerce com uma lista de produtos e carrinho de compras."
